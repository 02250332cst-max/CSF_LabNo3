# CSF_LabNo3
Aim of the Lab

The aim of this lab is to develop a clear understanding of control structures in Python. This includes learning how to use conditional statements, loops (for and while), and control statements such as break and continue. Additionally, the lab focuses on defining and invoking functions to solve programming problems in a structured, efficient, and modular way.
